## 📦 分享方式

### 方式1：分享完整项目

```bash
# 运行打包脚本
./package_project.sh

# 生成三个压缩包：
# 1. xiaohongshu_automation_full_[时间].zip - 完整项目
# 2. xiaohongshu_automation_docs_[时间].zip - 仅文档
# 3. xiaohongshu_automation_data_[时间].zip - 数据包
```

将生成的压缩包发送给对方即可。

### 方式2：Git仓库分享

```bash
# 初始化Git仓库
git init

# 添加所有文件（排除敏感信息）
git add README.md QUICKSTART.md API_DOCS.md
git add setup_all.sh package_project.sh
git add *.py
git add woshipm_ai_articles_detailed_v2.json

# 创建.gitignore
cat > .gitignore << 'EOF'
# 排除敏感文件
xiaohongshu-mcp/cookies.json
*.log

# 排除临时文件
*.pyc
__pycache__/
.DS_Store

# 排除大文件
articles_pdf/
packages_*/
EOF

# 提交
git commit -m "初始提交：小红书自动化系统"

# 推送到GitHub（需要先创建仓库）
git remote add origin [你的仓库地址]
git push -u origin main
```

### 方式3：网盘分享

1. 运行 `./package_project.sh` 打包
2. 将生成的压缩包上传到网盘
3. 分享下载链接

## 🎯 核心功能说明

### 功能1：内容爬取
**文件**：`woshipm_ai_crawler_v2.py`

**用途**：从"人人都是产品经理"爬取AI相关文章

**输出**：
- JSON格式（程序使用）
- CSV格式（Excel查看）
- PDF格式（阅读查看）

### 功能2：自动发布
**文件**：`auto_publish_rest.py`

**用途**：100%自动化发布到小红书

**特点**：
- 无需手动操作
- 自动生成配图
- 自动适配格式
- 40-60秒/篇

### 功能3：配图生成
**文件**：`create_professional_images.py`

**用途**：生成专业的小红书配图

**规格**：
- 尺寸：1080x1350
- 格式：JPG
- 质量：高清

### 功能4：批量发布
**文件**：`batch_publish.py`

**用途**：批量发布多篇文章

**功能**：
- 自动间隔控制
- 进度显示
- 错误处理
